Frequently Asked Questions

What are Open Graph meta tags?

Open Graph is a protocol created by Facebook that controls how URLs are displayed when shared on social media. It uses meta tags like og:title, og:description, and og:image to define the title, description, and image shown in social posts.

What's the difference between Open Graph and Twitter Cards?

Twitter Cards are Twitter's version of Open Graph. While Twitter can fall back to Open Graph tags, Twitter Card tags (twitter:title, twitter:description) let you customize how your content appears specifically on Twitter/X.

What image size should I use?

The recommended size is 1200x630 pixels (1.91:1 aspect ratio). This works well for Facebook, Twitter, and LinkedIn. Minimum size is 600x314 pixels. Use JPG or PNG format, and keep file size under 5MB.

How do I test my meta tags after adding them?

Use Facebook's Sharing Debugger, Twitter's Card Validator, or LinkedIn's Post Inspector to test how your page appears. These tools also clear the cache if you've recently updated your meta tags.
